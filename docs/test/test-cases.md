# Тест-кейсы

## Страница регистрации пользователя

### Регистрация пользователя

ID:

Заголовок: Регистрация пользователя

Предусловие: Открыта страница регистрации пользователя на английском языке

| Шаг                                                                                               | ОР                                                                      |
|---------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------|
| Ввести в поле "Username" имя пользователя, состоящее из английских букв, цифр, символов "_" и "-" | В поле "Username" отображается введенное имя                            |
| Ввести в поле "Email" адрес электронной почты                                                     | В поле "Email" отображается введенный адрес                             |
| Ввести в поле "Password" пароль пользователя (минимум 8 символов)                                 | В поле "Password" отображается введенный пароль в виде звездочек        |
| Ввести в поле "Repeat password" пароль из прошлого шага                                           | В поле "Repeat password" отображается введенный пароль в виде звездочек |
| Нажать на кнопку "Sign up" для завершения регистрации                                             | Регистрация пользователя завершается, открывается страница пользователя |

## Страница создания нового медиа

### Создание нового медиа

ID:

Заголовок: Создание нового медиа

Предусловие: Открыта страница создания нового медиа на английском языке, пользователь авторизован

| Шаг                                                                           | ОР                                                                                                                                                                  |
|-------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Ввести в поле "Title" название медиа на английском языке                      | В поле "Title" отображается введенное название                                                                                                                      |
| Внести или выбрать в поле "Release Date" дату релиза                          | В поле "Release Date" отображается введенная или выбранная дата                                                                                                     |
| Внести или выбрать в поле "Genres" жанры (один или несколько)                 | В поле "Genres" отображаются введенные или выбранные жанры                                                                                                          |
| Внести в поле "Original Title" оригинальное название медиа на языке оригинала | В поле "Original Title" отображается введенное оригинальное название                                                                                                |
| Внести или выбрать в поле "Media Type" тип медиа (один)                       | В поле "Media Type" отображается введенный или выбранный тип медиа                                                                                                  |
| Внести или выбрать в поле "Participants" компании, персоны и их роли          | В поле "Participants" отображаются введенные или выбранные элементы                                                                                                 |
| Добавить картинку через поле "Drag poster here"                               | Добавленная картинка отображается в поле "Drag poster here"                                                                                                         |
| Нажать на кнопку "Create" для завершения создания нового медиа                | Создание нового медиа завершается, открывается страница пользователя, созданное медиа привязывается к создавшему пользователю. Созданное медиа доступно через поиск |
