# Тест-кейсы

## Страница регистрации пользователя

### Регистрация пользователя

ID:

Заголовок: Регистрация пользователя

Предусловие: Открыта страница регистрации пользователя на английском языке

| Шаг                                                                           | ОР                                                                      |
|-------------------------------------------------------------------------------|-------------------------------------------------------------------------|
| Ввести в поле "Username" имя пользователя, состоящее из английских букв, цифр | В поле "Username" отображается введенное имя                            |
| Ввести в поле "Email" адрес электронной почты                                 | В поле "Email" отображается введенный адрес                             |
| Ввести в поле "Password" пароль пользователя (минимум 8 символов)             | В поле "Password" отображается введенный пароль в виде звездочек        |
| Ввести в поле "Repeat password" пароль из прошлого шага                       | В поле "Repeat password" отображается введенный пароль в виде звездочек |
| Нажать на кнопку "Sign up" для завершения регистрации                         | Регистрация пользователя завершается, открывается страница пользователя |

Предусловие: Открыта страница регистрации пользователя на английском языке, пользователь с таким именем пользователя и/или адресом электронной почты уже существует в системе

| Шаг                                                                           | ОР                                                                                                                                                                                                  |
|-------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Ввести в поле "Username" имя пользователя, состоящее из английских букв, цифр | В поле "Username" отображается введенное имя                                                                                                                                                        |
| Ввести в поле "Email" адрес электронной почты                                 | В поле "Email" отображается введенный адрес                                                                                                                                                         |

| Ввести в поле "Repeat password" пароль из прошлого шага                       | В поле "Repeat password" отображается введенный пароль в виде звездочек                                                                                                                             |
| Нажать на кнопку "Sign up" для завершения регистрации                         | Пользователь получает сообщение о том, что пользователь с таким именем и/или адресом электронной почты уже существует (соответствующие поля ввода выделяются красным и появляются подписи под ними) |

## Страница авторизации пользователя

### Авторизация пользователя

ID:

Заголовок: Авторизация пользователя

Предусловие: Открыта страница авторизации пользователя на английском языке

| Шаг                                                                           | ОР                                                                      |
|-------------------------------------------------------------------------------|-------------------------------------------------------------------------|
| Ввести в поле "Email / Username" адрес электронной почты или имя пользователя | В поле "Email / Username" отображается введенный адрес или имя          |
| Ввести в поле "Password" пароль пользователя                                  | В поле "Password" отображается введенный пароль в виде звездочек        |
| Нажать кнопку "Login" для завершения авторизации                              | Авторизация пользователя завершается, открывается страница пользователя |
