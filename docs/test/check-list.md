# Чек-листы тестирования

## Чек-лист тестирования страницы регистрации пользователя

1. Проверка ввода имени пользователя в соответствующее поле
2. Проверка ввода электронной почты в соответствующее поле
3. Проверка ввода пароля и повторение пароля в соответствующие поля (символы отображаются в виде звездочек)
4. Проверка кнопки переключения видимости пароля (в виде открытого/зачеркнутого глаза) в полях введения пароля и повторения пароля
5. Проверка кнопки регистрации
6. Проверка кнопки переключения языка (в виде глобуса)
7. Проверка переключения языка интерфейса на английский язык
8. Проверка переключения языка интерфейса на русский язык
9. Проверка переключения языка интерфейса на японский язык
10. Проверка кнопки переключения темы интерфейса (в виде солнца/луны)
11. Проверка корректности отображения светлой темы интерфейса
12. Проверка корректности отображения темной темы интерфейса
13. Проверка корректности отображения макета интерфейса на десктопном разрешении
14. Проверка корректности отображения макета интерфейса на мобильном разрешении

## Чек-лист тестирования страницы пользователя

1. Проверка кнопки выхода пользователя из аккаунта
2. Проверка кнопок переключения типов медиа (активный элемент помечается цветом)
3. Проверка строки поиска
4. Проверка кнопки создания нового медиа (в виде плюса)
5. Проверка корректного отображения таблицы медиа
6. Проверка переключения фильтрации по статусам
7. Проверка кнопки переключения языка (в виде глобуса)
8. Проверка переключения языка интерфейса на английский язык
9. Проверка переключения языка интерфейса на русский язык
10. Проверка переключения языка интерфейса на японский язык
11. Проверка кнопки переключения темы интерфейса (в виде солнца/луны)
12. Проверка корректности отображения светлой темы интерфейса
13. Проверка корректности отображения темной темы интерфейса
14. Проверка корректности отображения макета интерфейса на десктопном разрешении
15. Проверка корректности отображения макета интерфейса на мобильном разрешении
16. Проверка фильтров в таблице медиа
17. Проверка сортировок в таблице медиа
